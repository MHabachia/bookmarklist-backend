Repo für das Projekt des Kurses „Webtechnologien“, SoSe2026

Dieses Repo dient der Entwicklung einer Web-App zur Verwaltung von Lesezeichen. „Backend“
Die App wird den Nutzern grundlegend ermöglichen, Links mit Titeln und kurzen Beschreibungen in das System aufzunehmen.
Sollte sich der Umfang des Projekts erweitern, werden Funktionen zur Kategorisierung und Gruppierung der gespeicherten Einträge sowie eine Textsuchfunktion implementiert.
